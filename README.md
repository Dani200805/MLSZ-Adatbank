# MLSZ-Adatbank
MLSZ Adatbank

Az MLSZ Adatbank egy modern webalkalmazás, amely a magyar labdarúgás adatainak kezelésére és megjelenítésére készült.
A projekt célja, hogy a felhasználók könnyedén böngészhessenek csapatok, játékosok és mérkőzések között, miközben az alkalmazás teljes CRUD funkcionalitást biztosít REST API alapokon.

Funkciók:
  Csapatok listázása
  Játékosok megjelenítése
  Mérkőzések adatainak kezelése
  Új adatok hozzáadása
  Meglévő adatok szerkesztése
  Adatok törlése megerősítéssel
  Reszponzív felhasználói felület
  Hibakezelés és betöltési állapotok kezelése
  
Használt technológiák:
  Vite
  TypeScript
  HTML
  CSS
  JSON Server
  Git & GitHub
